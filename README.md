# Sasidhar Portfolio — Production Version

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Files included
- Local project images in `public/projects/`
- Local profile photo in `public/profile/`
- Local logo files in `public/logos/`

## Notes
- Tool, certification, company, and education logos are bundled locally.
- Some bundled logos are stylized local badge placeholders so everything works offline and loads reliably.
- You can later replace any SVG inside `public/logos/` with official SVG logo files using the same filenames.

## Resume and LinkedIn
The site is already wired to:
- LinkedIn profile
- Google Drive resume file

## Deploy
Upload the full folder contents to GitHub, then import the repo into Vercel.
